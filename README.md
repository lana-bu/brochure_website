# brochure_website
This is my Project #1 assignment for my CIS 435 (Web Technology) course at UM-Dearborn. I created it to learn how to implement responsive web design with HTML and CSS, ensuring the website appears clearly on any screen size (desktop, tablet, or mobile phone).
